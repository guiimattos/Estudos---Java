# Exercícios em Java para Estudos ☕

Este repositório contém exercícios simples em **Java**, com foco no aprendizado da lógica de programação, operadores e estruturas básicas da linguagem.

---

## Operadores em Java

Abaixo estão alguns operadores muito utilizados em Java:

| Operador | Nome | Descrição |
|---|---|---|
| `&&` | E | Retorna verdadeiro quando as duas condições forem verdadeiras |
| `||` | OU | Retorna verdadeiro quando pelo menos uma condição for verdadeira |
| `!` | NÃO | Inverte o valor lógico da condição |
| `==` | Igual | Verifica se dois valores são iguais |
| `!=` | Diferente | Verifica se dois valores são diferentes |
| `>` | Maior que | Verifica se um valor é maior que outro |
| `<` | Menor que | Verifica se um valor é menor que outro |
| `>=` | Maior ou igual | Verifica se um valor é maior ou igual a outro |
| `<=` | Menor ou igual | Verifica se um valor é menor ou igual a outro |
